🔥 Firebase Setup

One-time setup — both players need the same config. Paste your Firebase project config below.

📋 How to get your Firebase config (click to expand)
1. Go to console.firebase.google.com
2. Click Add project → name it anything → disable Analytics → Create
3. In your project, click the </> Web icon to register an app
4. Give it a nickname → click Register app
5. Copy the firebaseConfig object shown
6. In the left sidebar → Build → Realtime Database → Create database
7. Choose Start in test mode → Enable
8. Paste the config below and click Connect

Local Only = single player, saves to this browser only. Both players must use the same Firebase config to sync.

📐 Conditions Reference
Blinded: Auto-fail sight checks, attacks vs you have adv.
Charmed: Can't attack charmer; charmer adv on social checks.
Frightened: Disadv attacks/checks while source visible.
Grappled: Speed 0. Ends if grappler incapacitated.
Paralyzed: Incap, auto-fail Str/Dex, crits from 5ft.
Poisoned: Disadv on attack rolls and ability checks.
Prone: Disadv attacks; adv attacks vs from within 5ft.
Restrained: Speed 0, disadv attacks, adv attacks vs you.
